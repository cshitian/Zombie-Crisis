# 游戏设定说明 (Game Settings)

本文档记录了《僵尸危机》游戏中的各项核心设定，涵盖角色行为、系统逻辑及数值平衡。

## 1. 角色与阵营 (Roles & Entities)

### 1.1 平民 (Civilians)
- **个体差异**：包含成年男性、成年女性、儿童和老人。不同类型的平民在移动速度和基础属性上略有差异。
- **行为逻辑**：
  - **避难**：大多数平民（95%）倾向于留在室内。当感知到僵尸（视觉或听觉）时，会进入恐慌状态并在建筑间寻找掩体。
  - **流浪者**：极少数平民（5%）喜欢在户外游荡。
  - **武装化**：平民可以拾取掉落的武器。一旦武装，他们将尝试反击僵尸，甚至协助士兵。
- **状态波动**：拥有情绪气泡（Mood Bubbles）和内心独白（Thoughts），反映其当前处境（如恐慌、勇敢、饥饿等）。

### 1.2 僵尸 (Zombies)
- **行为逻辑**：
  - **空闲**：在没有猎物时缓慢游荡。
  - **追逐**：一旦通过嗅觉或听觉发现人类，会以爆发速度进行追击。
- **感染机制**：通过物理接触（近距离）进行感染。接触时间越长，感染风险越高。

### 1.3 士兵 (Soldiers / Spec Ops)
- **专业素质**：受建筑速度惩罚极小，拥有更高的基础血量和移动速度。
- **协同作战**：通过卫星频道共享目标位置，拥有更广的战术视野。
- **武器配备**：通常配备狙击步枪、散弹枪或喷网枪（捕获工具）。

### 1.4 医疗兵 (Medics)
- **特殊技能**：负责救治被喷网枪困住的僵尸。
- **治疗逻辑**：自动寻找并移动至被困目标，执行耗时约 5 秒的治疗程序，成功后可将僵尸转回平民。

## 2. 感知系统 (Perception System)

| 感知类型 | 范围 | 受建筑阻隔 | 适用角色 |
| :--- | :--- | :--- | :--- |
| **视觉 (Vision)** | ~200m (120°扇区) | 是 | 人类 (核心感知) |
| **听觉 (Hearing)** | ~90m (360°) | 否 | 全员 |
| **嗅觉 (Smell)** | ~500m (360°) | 否 | 僵尸 (核心感知) |
| **卫星感知 (Satellite)** | ~1000m | 否 | 专业人员 |

- **感知细节**：当建筑外部僵尸数量 > 3 时，室内人类会因听觉感知到威胁。

## 3. 武器与战斗 (Weapons & Combat)

| 武器名称 | 射程 | 伤害 | 特性 |
| :--- | :--- | :--- | :--- |
| **手枪 (Pistol)** | 中短 | 中等 | 基础防身武器 |
| **散弹枪 (Shotgun)** | 极短 | 高 | 范围伤害，近战杀器 |
| **狙击枪 (Sniper)** | 极长 | 极高 | 附带冷却时间，需保持射程距离 |
| **火箭筒 (Rocket)** | 中等 | 极高 (溅射) | 弹药有限，拥有智能避开友方的逻辑 |
| **喷网枪 (Net Gun)** | 中等 | 0 | 困住僵尸，配合治疗使用 |

## 4. 外部支援与任务 (Support & Tasks)

- **空投补给 (Supply Drop)**：在特定区域投放武器，武装当地平民。
- **特种突击 (Spec Ops)**：部署高精尖士兵执行清剿或营救任务。
- **空中打击 (Airstrike)**：对目标区域进行毁灭性轰炸。轰炸后会留下弹坑（Crater），清理范围内的所有目标。
- **战术分析 (Tactical Analysis)**：由 Gemini AI 提供，针对特定建筑生成实时战术简报和生存指南。

## 5. 环境与地理 (Environment)

- **建筑类型**：
  - **住宅 (Residential)**：平民的主要聚集区。
  - **商业/工业 (Commercial/Industrial)**：更有可能发现高级资源。
  - **公共建筑 (Public)**：如医院、政府机构等。
- **搜刮机制 (Scavenge)**：玩家可指派平民或士兵搜刮建筑资源，不同类型的建筑提供不同比例的资源奖励。
